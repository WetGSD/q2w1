Responsive Web Design

Responsive Web Design solves the problem of displaying web content well on all devices.

Fixed web page designs for a desktop or laptop do not scale well to a phone or a tablet such that the site may be avoided by customers with phones or tablets.

Adaptive design was used for particular screen sizes, but responsive design is meant to change the web page appearance and adjust to the size of the screen or with resizing of the browser window.

Responsive Web Design is using HTML and CSS to make web pages look good on all devices.

Pages are designed first for smaller screens and include all the required fields and content.

HTML and CSS are used to resize, shrink or enlarge content to fit any screen.
